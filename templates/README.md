# Project template README

Use this README as a starting point for new projects inside `agents/projects/<project-name>`.

Replace the placeholders below:

- Project name: <project-name>
- One-line description: Brief summary of what this agent/project does.
- Owner: Your name or team.

## Quick start

1. Fill `project.json` metadata.
2. Edit `agent_spec.md` with goals and prompts.
3. Add source code in `src/` and assets in `assets/`.

## License

Specify the license in the `LICENSE` file (SPDX identifier in `project.json`).
